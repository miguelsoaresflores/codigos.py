Lógica de Programação — Python
Repositório criado para armazenar os programas desenvolvidos durante a disciplina de Lógica de Programação do curso de Engenharia de Software no CEUB.
Os exercícios aplicam conceitos fundamentais de programação como condicionais, laços de repetição, funções e operações matemáticas.

Programas
📊 Média do Aluno com Exame
Programa que calcula a média de um aluno e verifica se foi aprovado, enviado para exame final ou reprovado.
Aplica condicionais e cálculo de médias ponderadas.
🚗 Sistema de Estacionamento
Programa que calcula o valor a pagar em um estacionamento com base no tempo de permanência do veículo.
Aplica estruturas condicionais e operações matemáticas com horas e frações de tempo.
🌡️ Conversor de Temperatura
Programa que converte temperaturas entre Celsius, Fahrenheit e Kelvin.
Aplica condicionais e fórmulas matemáticas de conversão.
